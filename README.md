## SocIO Chat
### SocketIO based Chat (공부겸 빌딩중)
#### 사용언어: html,css,VanillaJS 및 node.js express,socket.IO 
### Feature
<ul>
    <li>채팅 방 생성</li>
    <li>유저 회원가입/로그인/로그아웃</li>
    <li>친구 추가(구현중)</li>
    <li>채팅방에서 메시지 입력</li>
    <li>유저 채팅방 입장 혹은 나감 통지가능</li>
</ul>
