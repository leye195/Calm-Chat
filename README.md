# Calm Chat⛺️🌧

켐프에서 시원한 빗소리를 들으며 서로의 이야기를 공유하는 컨셉의 익명 대화방 입니다.

### 활용 기술 및 언어:

- html,css,VanillaJS,
- node.js express,socket.io

### 사용 방법

#### 환경변수 env

.env 안에 port를 설정해주세요

```
PORT = ${PORT_NUMBER}
```

#### NPM

- 설치: npm install

- 실행: npm run dev:server
